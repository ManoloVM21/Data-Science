# Data Science Portfolio Repository

This repository contains a collection of data science challenges and end-to-end machine learning projects.

## Repository Structure

### Challenges
- **Data Cleaning** (`CHALLENGES/Data Cleaning/`)
  - Practical exercises focused on preparing real-world datasets.
  - Includes missing value handling, data normalization, and transformation workflows.
- **Data Manipulation and Visualization** (`CHALLENGES/Data Manipulation and Visualization/`)
  - Exercises for exploratory analysis and reporting.
  - Covers dataframe operations and visual storytelling for insights.

### Full Projects
- **House Price Predictions - XGBoost**
  - Supervised regression project for property value estimation.
  - Focus on feature engineering and gradient boosting models.

- **Bike Rentals per Hours - Neural Networks & XGBoost**
  - Time-based demand prediction project.
  - Combines deep learning and tree-based boosting approaches.

- **CV_admitter**
  - Project artifacts related to candidate profile analysis.
  - Includes a skills mapping file used for model or rule-based processing.

- **Bank Loan Program - Random Forest**
  - Classification workflow to support lending-related decisions.
  - Uses ensemble learning and structured data preprocessing.

- **forecastventas**
  - Sales forecasting project with notebooks and a Streamlit application.
  - Includes training/inference workflows and an interactive simulation dashboard.
  - **Important:** `forecastventas` was a **guided project**.

## Skills Demonstrated Across the Repository

- Data cleaning and preprocessing
- Exploratory data analysis (EDA)
- Data visualization and communication
- Feature engineering
- Supervised learning (regression and classification)
- Ensemble methods (Random Forest, XGBoost)
- Neural networks for predictive modeling
- Time-aware forecasting workflows
- Model inference pipelines
- Interactive app prototyping with Streamlit

## Techniques and Tools Used

- Python-based data science stack (pandas, NumPy, matplotlib, seaborn)
- Notebook-driven experimentation
- Model serialization and loading workflows
- Dashboard-style model simulation interfaces
- Structured project organization for challenges vs. full implementations

## Notes

This repository is intended as a practical learning and portfolio space, showing progression from foundational data tasks to complete machine learning applications.
